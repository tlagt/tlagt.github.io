2.\n & endl 差异

#include
#include <unistd.h>

using namespace std;

int main() {
cout << “即将开始等待…” << endl;
cout << “等待中…” << endl;
sleep(5);
cout << endl;
cout << “等待结束。” << endl;
return 0;
}
若将等待中之后的endl替换为\n，输出结果不同，等待中将会在5秒后才出现

这是由于endl确保程序继续运行前刷新输出
